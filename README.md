# Projeto de Segurança da Informação
Este projeto implementa um sistema de autenticação utilizando JWT, com foco em boas práticas de segurança.

## Funcionalidades
- Cadastro de usuários com senha criptografada (BCrypt)
- Login com geração de token JWT
- Validação de rotas protegidas
- Persistência de dados em JSON

## Tecnologias utilizadas
- Java
- JWT
- BCrypt
- JSON

## Estrutura do projeto
- Backend: API de autenticação
- Frontend: interface de login e dashboard
